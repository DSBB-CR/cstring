# C String Library (`str.h` / `str.c`)

Простейшая библиотека для работы со строками в C, реализующая динамический строковый тип `String`. Предоставляет удобные функции для создания, управления, сравнения и ввода/вывода строк.

## Особенности

- Динамическое управление памятью.
- Безопасное копирование и конкатенация.
- Функции для чтения из потоков (`stdin`, файлы).
- Изменение регистра символов.
- Срезы строк (substrings).
- Простой интерфейс, похожий на стандартные строковые функции C.

## Структура `String`

```c
typedef struct string {
    char* str;   // Указатель на массив символов
    size_t len;  // Длина строки
} String;
```
